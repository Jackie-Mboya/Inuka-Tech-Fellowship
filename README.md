# Inuka Tech Fellowship

Welcome to my official repository for the **Inuka Tech Fellowship**. This space serves as a centralized portfolio documenting my technical growth, data engineering pipelines, operational analytics projects, and data governance insights compiled throughout the fellowship program.

---

## 📌 Program Overview
The Inuka Tech Fellowship is an intensive, practical training program designed to equip emerging data professionals with advanced skills in scalable data architectures, statistical modeling, and strategic data communication. 

* **Track:** Data Science & Analytics
* **Core Focus:** Building end-to-end data pipelines, developing reproducible analytics workflows, and translating complex data metrics into actionable insights for leadership.

---

## 🛠️ Technical Stack & Tooling
Throughout this fellowship, I leverage an advanced, industry-standard stack to solve high-impact operational challenges:

* **Languages:** Python (Pandas, NumPy, Scikit-learn), SQL (PostgreSQL)
* **Data Engineering & Orchestration:** Apache Airflow, Apache Kafka, Apache Spark, Docker
* **Business Intelligence:** Power BI, Markdown / HTML Reporting
* **Version Control & DevOps:** Git & GitHub
